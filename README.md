Submitted by: **Hafsa Konain**

## Overview

AeroPredict is an iOS mobile app that helps users search flights and view delay predictions, risk insights, and saved flight tracking.

---

## App Evaluation

### Mobile
This app is designed specifically for mobile users who need quick access to flight information on the go.

### Story
Travelers often do not know whether a flight will be delayed until the last moment. AeroPredict helps users make better travel decisions by predicting delays in advance.

### Market
The app targets travelers, students, and frequent flyers. It has a large potential user base.

### Habit
Users can open the app before traveling or while planning trips to check predictions.

### Scope
The MVP includes search, prediction, saved flights, and dashboard. Additional features can be added later.

---

## Features

### Required Features
- User can search for a flight by flight number
- User can view delay prediction and risk level
- User can save/bookmark flights

### Optional Features
- User can view dashboard insights and airline trends
