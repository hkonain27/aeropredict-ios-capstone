# AeroPredict

Submitted by: **Hafsa Konain**

## Overview
AeroPredict is an iOS mobile app that helps users search flights and view delay predictions, risk insights, and saved flight tracking.

## Features
- Search for flights
- View delay prediction
- View risk level
- Save/bookmark flights
- Dashboard insights

## Screens
- Home Screen
- Prediction Screen
- Saved Flights Screen
- Dashboard Screen
- Profile Screen

## Navigation
Tab Bar:
- Home
- Saved
- Dashboard
- Profile
